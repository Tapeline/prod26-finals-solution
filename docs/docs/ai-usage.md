# Политика использования генеративного ИИ

## Рамки использования ИИ

Генеративный ИИ используется в рамках данного проекта для:

- поиска и исследований (Deep Research) наряду с классическими инструментами поиска
- трансформации представления материалов (например, отрисовки компонентной диаграммы
  по написанной спецификации и ADR или форматированием результатов исследования в виде ADR)

Несмотря на это, любой сгенерированный отрывок вручную отчитан и провалидирован.

## Перечень сгенерированных материалов

Это перечень материалов полностью или в значительной степени 
состоящих из сгенерированного контента.

- Компонентная диаграмма C4
- ADR009 — JSON Schema в качестве формата схемы событий
