# Insights UI

## Сущности

Не вводит новых сущностей, просто предоставляет удобный интерфейс визуализации

## Сценарии

### Experiment Insights

**Кто**: любой пользователь

Опционально принимает на вход фильтрацию по атрибутам событий в виде GET параметров
(например, `?country=RU&version="1.3.0"`)

**Что отображает**:

- долю отклонённых событий и причины (топ причин);
- долю дубликатов;
- долю событий конверсии/ошибок/задержки, степень полноты атрибуции
  ??? как определять, если все события определяются пользователем ???
- задержку доставки событий — если собирается;
- перекос трафика: фактическое распределение показов по вариантам против ожидаемого по весам (как предупреждение).

### Experiment Insights CSV Export

**Кто**: любой пользователь

**Что экспортирует**:

Экспорт CSV будет отдавать ZIP файл с такими CSV:

- CSV причин отклонения:

    | Причина | Кол-во отклонённых |
    |---------|--------------------|
    |         |                    |

- CSV по вариантам

    | Вариант | Сколько пользователей получило | Ожидаемая доля распределения | Реальное распределение | Сколько событий отклонено | Сколько событий принято |
    |---------|--------------------------------|------------------------------|------------------------|---------------------------|-------------------------|
    |         |                                |                              |                        |                           |                         |
