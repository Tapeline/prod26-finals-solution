# Insights UI

## Сущности

Не вводит новых сущностей, просто предоставляет удобный интерфейс визуализации

## Сценарии

### Experiment Insights

**Кто**: любой пользователь

Опционально принимает на вход фильтрацию по атрибутам событий в виде GET параметров
(например, `?country=RU`)

**Что отображает**:

- pie chart по статусам событий (отклонён/дубликат/ожидает атрибуцию/принят)
- bar chart по типам событий
- pie chart по причинам отклонения
- gauge полноты атрибуции (`(принято / (принято + ожидают атрибуцию)) * 100%`)
- p50, p75, p95 задержки доставки
- pie chart реального распределения показов по вариантам
- таблица отклонений от ожидаемого распределения по вариантам
- предупреждение о слишком большом отклонении в распределении
