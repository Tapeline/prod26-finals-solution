## Сценарии

### Статистика конфликтов по эксперименту

**Кто**: любой пользователь

**Шаги**:

1. Запрашивается агрегированная статистика конфликтов эксперимента: сколько раз и по каким политикам
   применялся и отклонялся

### Статистика конфликтов по домену

**Кто**: любой пользователь

**Шаги**:

1. Запрашивается агрегированная статистика конфликтов домена: 
   сколько раз конфликтовал тот или иной эксперимент


!!! note
    "Сколько раз конфликтовал", "количество конфликтов" считаются не по одному
    на принятое решение, а по одному на конфликтующий эксперимент. Фактически,
    считаются факты разрешения конфликта, то есть решения по каждому из
    конфликтных экспериментов. В таком случае, если в домене A столкнулись
    эксперименты B, C и D, то конфликтов (записей о разрешении) будет 3
    (для B, для C и для D).
