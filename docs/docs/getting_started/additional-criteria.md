## Insights

Проверяются автотестами: `sys-tests/tests/f02`.

Чтобы наполнить содержимым, предлагается запустить тесты из `sys-tests/tests/scenario`:

```sh
cd sys-tests
uv sync
uv run pytest tests/scenario
```

После прогона можно посмотреть глазами на фронтенде:

- открыть `{адрес сервера}/_iap/login` в браузере:
- войти с заготовленным аккаунтом администратора
- Experiments -> эксперимент -> Insights
- или по постоянной ссылке вида `{адрес сервера}/#experiment/{id эксперимента}?tab=insights`

!!! note
    Фронтенд кроме Insights не заявляется как фича, он не доделан и нужен был мне
    только для более простых тестов. Невозможность сделать что-л. через UI
    не означает невозможность бэкенда обработать запрос.
