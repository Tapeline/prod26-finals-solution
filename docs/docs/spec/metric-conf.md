# Описания выборки метрик

Каждая метрика имеет:

- фильтр на события
- функцию агрегации по этим событиям

## Фильтр на события

Описывается строкой, содержащей в себе одно или несколько <evt:ID Типа> события,
разделённых `;`.

Например:

- `buttonClicked`
- `buttonClicked;screenSwiped`

## Функция агрегации

Вычисляет значение метрики. Доступны следующие функции:

- `sum` — Сумма
- `avg` — Среднее
